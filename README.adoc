= REST-API

.features:
* Signup account
* Password reset and update
* Account settings
* Users management

== Running

=== **Test the app**

Open terminal and run :
[source,bash]
----
./gradlew check
----

=== **Run the app**

Open terminal and run :
[source,bash]
----
./gradlew bootRun
----

and launch link:http://localhost:8880/[http://localhost:8880/]

=== **to run the cli**

Open terminal and run :
[source,bash]
----
./gradlew cli
----

=== **to run firefox and show test reports(required firefox installed)

Open terminal and run :
[source,bash]
----
./gradlew :apiCheckFirefox
----
